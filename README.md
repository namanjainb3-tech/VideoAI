# 🚀 Overview

VideoAI is an **AI-powered creator platform** that automates one of the most repetitive parts of content creation—**video editing**.

Instead of manually switching between multiple software applications, creators can upload a single video and let VideoAI automatically perform the entire editing pipeline.

The platform combines **computer vision**, **speech recognition**, **large language models**, and **video rendering** into one seamless workflow.

Whether you're creating Instagram Reels, YouTube Shorts, educational videos, podcasts, or marketing content, VideoAI dramatically reduces editing effort while maintaining professional output quality.

---

# 🎯 The Problem

Modern creators often use multiple tools during a single editing session.

Typical workflow:

- Record the video
- Remove or blur the background
- Generate subtitles
- Correct caption mistakes
- Highlight important words
- Merge branded intros
- Export final video
- Ask ChatGPT whenever help is needed

This requires switching between multiple applications and often paying for premium features.

---

# 💡 Our Solution

VideoAI brings all of these capabilities together into a **single AI-powered platform**.

Simply upload your video once.

The platform automatically performs:

- 🎥 AI Background Blur
- 🎙 Speech-to-Text
- 💬 Subtitle Generation
- 🟨 Word-by-Word Caption Highlighting
- 🤖 AI Caption Review
- 🎬 Intro Video Merging
- 📥 Final Video Rendering
- 🧠 Built-in AI Assistant (Kitu)

Everything happens inside one unified workflow.

# ⚡ AI Pipeline

VideoAI combines multiple AI systems into a single creator workflow.

| AI Component | Purpose |
|--------------|----------|

# 🚀 Getting Started

Follow these instructions to run VideoAI locally.

---

# 📦 Clone the Repository

```bash
git clone https://github.com/namanjainb3-tech/VideoAI.git

cd VideoAI
```

---

# ⚙ Backend Setup

## Create Virtual Environment

### Windows

```bash
python -m venv .venv

.venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv .venv

source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create Environment Variables

Create a file named:

```text
.env
```

Add your API keys:

```env
VIDEO_GROQ_API_KEY=your_video_groq_key

KITU_GROQ_API_KEY=your_kitu_groq_key
```

---

## Start Backend

```bash
python main.py
```

Backend runs on

```
http://localhost:5000
```

---

# 🎨 Frontend Setup

Open another terminal.

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on

```
http://localhost:5173
```

# 📡 REST API

## Home

```http
GET /
```

Returns backend status.

---

## Process Video

```http
POST /process
```

Uploads a video and starts the complete AI pipeline.

---

## Job Status

```http
GET /status/{job_id}
```

Returns

- Processing stage
- Progress
- Review information
- Download URL

---

## Caption Review

```http
POST /review
```

Accepts manual caption corrections and regenerates the final output.

---

## Download

```http
GET /download/{job_id}
```

Downloads the rendered MP4 video.
